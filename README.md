# drama-workflow

漫剧工业化工作流 v4.4

从剧本文本到 Block分镜/六宫格/Keyframes 的全链路离线流水线。

## 特性

- 结构化 Schema 契约 + 连续性协议 + 参考角色分离
- 15 Stage Pipeline 编排
- 7层 Prompt 编译体系
- 护栏规则引擎 (19条)
- 视频前置校准层
- 多候选视觉 QA
- 经验库回写

## 快速开始

```bash
python3 scripts/verify_full_chain.py
```

## 模块清单

- `adapters/` — 34个 Python 模块
- `scripts/` — 4个执行脚本
- `references/` — 27个参考文档
